JS : How to pass data to an anchor element

If you want to insert an <a></a> element using JavaScript, and you need to pass also the href and the content of the element, you can use the innerHTML property of the parent element and the template literals feature of JavaScript. Template literals are strings that can contain expressions and variables inside curly braces and are enclosed by backticks. For example, if you have a variable called link that contains the href value, and a variable called text that contains the content of the element, you can use the following code to insert the <a></a> element: 

            // Get the parent element where you want to insert the <a></a> element
            var parent = document.getElementById("parent");
            // Use innerHTML and template literals to insert the <a></a> element
            parent.innerHTML = <a href="${link}"> ${text} </a>;
This code will create an <a></a> element with the href attribute set to the value of the link variable, and the content set to the value of the text variable.
